# DataAcqDemo
实时信号采集Demo
基于Qt5，开发实时信号采集Demo，要求采样率51.2K，通道数32个，并实时处理、可视化和持久化数据。
系统核心功能要求如下：
实时采集：
模拟一个高速数据源（如 QTimer 或独立线程）按采样率和通道数产生数据。
高性能数据处理：
实现实时滑动窗口均值滤波（窗口大小可配置，如 100ms），支持在线峰值检测（如超过阈值时触发告警），处理延迟需控制在 < 5ms（从采集到处理完成）
可视化展示：
使用 QCustomPlot 或 Qt Charts 实时绘制 1~4 个通道的波形，通道可切换，图形更新频率不低于 20 FPS，且不能因数据量大而卡顿或丢帧。
数据持久化：
将原始数据异步写入 CSV 文件，避免阻塞主线程。支持暂停/恢复写入，防止数据丢失。文件按时间分段（如每 5 分钟一个文件）。
用户控制界面：
提供“开始”、“停止”按钮，可动态调整采样频率、滤波参数、告警阈值等。
